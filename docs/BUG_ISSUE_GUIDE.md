# 버그 발견 시 Issue 작성 안내

프로젝트 사용 중 **버그를 발견한 경우**,  
구두 전달이나 개인 메시지 대신 **GitHub Issue로 등록**해주시기 바랍니다.

## 🐛 Issue 등록이 필요한 이유
- 버그 이력을 체계적으로 관리하기 위함
- 재현 조건 및 해결 상태를 공유하기 위함
- 누락 없이 우선순위를 정해 처리하기 위함

## ✅ Issue 등록 방법
1. 해당 GitHub Repository 접속
2. **Issues** 탭 클릭
3. **New Issue** 선택
4. Bug Report 템플릿 선택
5. 템플릿 양식에 맞추어 작성

## ⚠️ 유의사항
- 단순 문의나 의견은 Issue가 아닌 Discord 채팅 채널을 이용해주세요.
- 중복 Issue가 있는지 먼저 확인 후 등록 바랍니다.
